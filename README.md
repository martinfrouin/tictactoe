# tictactoe

A Tic Tac Toe game made with Flutter and Rive.

## Requirements
- Flutter 3.27.x
- Dart 3.6.x

## Getting Started

### Install dependencies
```dart
flutter pub get
```

### Run the app
```dart
flutter run
```

### Run tests if needed
```dart
flutter test
```

## Known Improvements
- The GHA are just building Android and Web apps because of Apple's restriction for iOS and MacOS.
